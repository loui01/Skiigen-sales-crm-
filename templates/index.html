Where is my data stored?

Submitted registrations are stored in a local SQLite database file located at data/users.db. SQLite is a self-contained, serverless database that is perfect for prototyping or local demos.

When you're ready to move to a managed database such as PostgreSQL, you can reuse the same schema and migrate the data using export tools (e.g., sqlite3's .dump command).

View the current records by requesting /users. Protect this endpoint before deploying to production.
